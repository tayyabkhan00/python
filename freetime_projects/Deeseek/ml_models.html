How the Model Works

Step-by-Step Computation

Step 1: Input Features

The model receives input values X₁ and X₂

Step 2: Multiply by Weights

Each input is multiplied by its corresponding weight

X₁ × W₁ + X₂ × W₂

Step 3: Add Bias

The bias term is added to the weighted sum

Weighted Sum + b

Step 4: Activation Function

The result passes through an activation function to produce the final output

Output = Sign(Weighted Sum + b)

Model Parameters

Weights (W₁, W₂)

Weights determine the importance of each input feature. Positive weights increase the output, negative weights decrease it.

Bias (b)

Bias allows the model to shift the decision boundary. It's like the intercept in a linear equation.

Activation Function

This function converts the weighted sum into a useful output. Different models use different activation functions.
